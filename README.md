TrafficRNNTrain.py（云端训练代码）
基于 PyTorch 搭建 RNN/GRU/LSTM 统一训练框架，加载加州 PeMS 车流时序数据完成单步预测训练，内置信号捕获断点续训功能，训练结束输出轻量化模型权重文件，适配共享 GPU 环境。
TrafficServer.py（边缘推理服务代码）
依托 WebSocket 搭建本地边缘实时推理网关，加载训练权重逐帧预测车流，模拟设备运算与网络传输延迟，以 JSON 格式持续向前端推送真实流量、预测流量、误差、延迟数据。
traffic_monitor.html（D3 可视化前端页面）
暗黑工业风监控示波器，自动连接 WebSocket 数据流，用实线展示真实车流、虚线绘制预测车流，同步实时展示误差与推理延迟，断线自动重连，直观呈现三类模型拟合、相位滞后、欠拟合等实验现象。
